## Description
AI-Doctor is an AI-driven application that analyzes symptoms and offers potential diagnoses and treatment recommendations. It aims to assist users in understanding their health conditions and taking preventive measures.

## Features
- **Medical Diagnosis:** Analyze symptoms and provide potential diagnoses.
- **Recommendations:** Offer treatment suggestions and preventive measures.
- **User-Friendly Interface:** Easy to use and navigate.

## Technologies Used
- **Python:** Core programming language.
- **Django:** Web framework.
- **HTML/CSS:** Frontend design.
